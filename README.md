# ❤️ Low-Cost ECG Monitoring System

A low-cost embedded health monitoring system developed using **NodeMCU (ESP8266)**, **Arduino Uno**, the **AD8232 ECG Sensor**, and a **Pulse Sensor**. The project acquires real-time ECG signals and heart rate data for basic health monitoring.

## 📌 Features

- Real-time ECG signal acquisition using the AD8232 ECG sensor
- ECG waveform displayed using the Arduino IDE Serial Plotter
- Heart rate (BPM) measurement using a Pulse Sensor
- BPM displayed on an OLED display
- Low-cost and portable embedded healthcare solution

## 🚀 Working

1. The AD8232 ECG sensor is connected to the NodeMCU.
2. ECG signals are acquired and displayed using the Arduino IDE Serial Plotter.
3. The Pulse Sensor is connected to the Arduino Uno.
4. Heart rate (BPM) is calculated and displayed on the OLED display.
