# Low-Cost ECG Monitoring System

## Overview

This project is a low-cost embedded healthcare system developed using NodeMCU (ESP8266), Arduino, the AD8232 ECG sensor, and the MAX30102 pulse oximeter.

The system acquires ECG signals through the AD8232 sensor and displays them on the Arduino IDE Serial Monitor. Heart rate and SpO₂ are measured using the MAX30102 sensor and displayed on an OLED display.

## Features

- Real-time ECG signal acquisition
- Heart rate monitoring
- SpO₂ (Blood Oxygen) monitoring
- OLED display output
- Serial Monitor ECG visualization
- Low-cost healthcare prototype

## Hardware Used

- NodeMCU (ESP8266)
- Arduino UNO
- AD8232 ECG Sensor
- MAX30102 Pulse Oximeter
- 0.96" OLED Display
- Jumper Wires
- Breadboard

## Software Used

- Arduino IDE
- Embedded C
